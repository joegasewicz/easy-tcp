# Easy TCP
Python WSGI Server written in C

### Compatibility
Will run on UNIX like systems only

### Quick Start

##### Install
```bash
    pip install east-tcp
```

##### Start Easy TCP Server
```bash
easy-tcp --port 7070 --max_processes 15 --app app.py --function start_app
```
